# 4. Software Requirements

## Functional Requirements
- Extract structured data from ruhsat images using LLM (OpenAI Vision API).
- Store and retrieve data in a memory/database interface.
- Download and parse web page HTML.
- Dynamically map data fields to web form inputs using LLM.
- Automate web form filling and submission.
- Support multiple web forms and data types.

## Non-Functional Requirements
- Modular, decoupled, and testable architecture
- Minimal maintenance for web page changes
- High test coverage and automated tests
- Clear documentation and code comments
