# 6. Data Interfaces & Variable Definitions

## Data Flow
1. license_llm extracts data from image → JSON (e.g., OCR_SAMPLE format)
2. webbot downloads HTML → memory
3. license_llm maps JSON fields to HTML inputs → mapping JSON
4. webbot fills form using mapping JSON

## Key Data Structures
- OCR_SAMPLE (dict):
  - tckimlik, dogum_tarihi, ad_soyad, plaka_no, belge_seri, sasi_no, model_yili, tescil_tarihi
- Mapping JSON (dict):
  - field_mapping: {data_field: html_selector}
  - actions: ["click#button_text"]

## Example Variables
- ruhsat_json: Extracted data from license_llm
- html_string: Downloaded HTML content
- mapping_json: Output from LLM mapping
